OpenJunto

Multi-agent coordination through structured deliberation, inspired by Benjamin Franklin's Junto (1727).

What it is

OpenJunto is a Claude Code plugin that turns the assistant into a coordinated team of 16 expert sub-agents — Product Manager, Distinguished Engineer, Security Engineer, SRE, Data Architect, Technical Writer, and more. A Manager agent triages each request, selects the relevant experts, and routes the work through mandatory peer review.

The point is the review. Out of the box, a single agent produces work that looks coherent because it optimizes for internal consistency, not for adversarial critique. OpenJunto forces a different expert to challenge each result rather than agree with it — testing failure modes, naming the strongest counterargument, and surfacing what the first pass missed.

It is built for senior engineers and tech leads doing production work who want a structured second opinion without holding sixteen perspectives in their own head. It is slower and costs more tokens than baseline Claude; that tradeoff is the product. See WHY.md for the honest version, including when it is not worth it.

Complexity triage
Simple / Moderate / Complex tiers keep process weight proportional to risk.
Adversarial review
All Moderate and Complex work is reviewed by an expert from a different domain.
Strongest objection & falsifier
Every recommendation carries its best counterargument and a named condition under which it would break in production.
Circuit breakers
Work auto-escalates to you after 3 revision cycles or 2 hours without meaningful progress.

Get started

The plugin lives in oj-claude. Install it from the marketplace inside a Claude Code session:

/plugin marketplace add openjunto/oj-claude
/plugin install oj@openjunto

Then start a new session (restart or /clear) to load it. OpenJunto activates automatically — describe the work in plain language; you do not invoke experts by name.

Repositories

  • juntospec The specification corpus — the genome. Defines what to generate; contains no generated code.
